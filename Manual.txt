1. Compilar en la consola dentro del proyecto npm install
2. Para correr el proyecto ejecutar escribir: npm run dev
3. Para acceder a las rutas api ingrese en el navegador
#ingresos
http://localhost:4000/api/getListadoExpIngresos/2022-05-23/2022-08-10
http://localhost:4000/api/getListadoIngresoMensualxTipRecurso/101/2022-05-23/2022-08-10
http://localhost:4000/api/getListadoIngresoMensualxCorteProced/101/2022-05-23/2022-08-10

#programados
http://localhost:4000/api/getListadoProgramaciones/2022-05-23/2022-08-10
http://localhost:4000/api/getListadoProgramacionesPonente/101/2022-05-23/2022-08-10
http://localhost:4000/api/getListadoProgramacionesFirmadoPonente/101/2022-05-23/2022-08-10

#escritos
http://localhost:4000/api/getListadoEscritosAnual/2022-05-23/2022-08-10
http://localhost:4000/api/getListaTipoEscritos/101/2022-05-23/2022-08-10
http://localhost:4000/api/getListadoEscritosPendienteAtendido/2022-05-23/2022-08-10

En la cabecera de la peticion http escribir los header:
headers: 
    authorization : <token>
